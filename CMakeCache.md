# CMakeCache

CMakeCache.txt 은 텍스트 파일이며 CMake 빌드를 수행할 때 빌드 경로에 생성됩니다. CMake에서 활용되는 변수들을 기록해놓습니다. 첫 빌드시 사용되는 변수 리스트를 전부 CMakeCache.txt에 기록하기 때문에 두 번째 빌드부터는 변수를 기록하는 작업을 반복할 필요가 없게됩니다.
